## 📖 Overview

This project is a complete motherboard design for **gesture-controlled interactive devices**. It integrates an ESP32C3 microcontroller, MEMS attitude sensor, OLED display, and advanced power management system. The board is designed for **balance games** where the device's orientation (detected via MEMS) controls on-screen elements.

**Key Applications:**
- Interactive gaming peripherals
- Motion-sensing control systems
- Battery-powered handheld electronics
- Educational IoT prototyping

---

## 🚀 Key Features

| Feature | Specification |
|---------|---------------|
| **MCU** | ESP32C3 (RISC-V core, Wi-Fi + BLE) |
| **Display** | OLED-LCD (I2C/SPI interface) |
| **Sensor** | High-precision MEMS (Analog Devices / Bosch) |
| **Power Management** | USB-C & Li-ion Battery PowerPath with Ideal Diode |
| **Low Power Mode** | **5 µA sleep current** |
| **PowerPath** | Seamless USB ↔ Battery switching even at low battery voltage |
| **Design Tool** | EasyEDA (Pro) |
| **Manufacturing** | Optimized for JLCPCB |
